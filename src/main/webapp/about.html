Purpose is to

  • Allow users to create new chat rooms with a unique code.
  • Allow users to join existing chat rooms with unique codes.
  • Show a list of current existing chat rooms each having their unique code.
  • Show all the users that message in the chat room to all participants.
  • Welcome users into the chat room.

History

  • To view the chat history you must follow these steps
  • When you first enter the website, you must generate a unique room code
  • You must remember that unique room code
  • Once you are done with the chat room, you may either generate a new room code or navigate into the about page
  • if you have chosen to generate a new room code, simply enter the room code of the chatroom history you want to see
  • if you have chosen to enter the about page, you can click back into Chat Rooms
  • Once you are back into Chat Rooms, enter the same room code, and it will display the chat room history
